# SettlementState

Status of the settlement:

State | Description 
---------|----------
 `created` | Created but not exported 
 `exported` | Exported 
 `failedToExport` | Tried to export but failed 



## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `Created`        | created          |
| `Exported`       | exported         |
| `FailedToExport` | failedToExport   |
